Forecast Accuracy Items

Forecast Accuracy Items GSheet is a process implemented by the NSC RAN Planning team for the Forecast Accuracy Tableau Dashboard. Using this GSheet, the Demand Planners can customize the Forecast Months for which specific Common IDs should be reported on the Forecast Accuracy Tableau Dashboard.

There are two distinct values in the `Input` tab's Month columns,

1. Exclude
These values have been preset by the logic using the `Planned Start Date` and `Planned End Date` columns, which are managed by the Tracked Items table process. These values cannot be edited via this GSheet. To edit these values, you will need to edit the Planning dates accordingly via the Tracked Items process managed by Terrence Belmonti.

2. Manual Constraint
These values have been manually edited and managed by the Demand Planners for their respective items via this GSheet. To edit these values, you can utilize the following two scripts available under the Editor menu,

  1. Add Constraint Items
    Use this option to add the `Manual Constraint` values to the specific Common IDs and Months. You can add single or multiple items and months. You can only add constraint items to current and future months.

  2. Remove Constraint Items
    Use this option to remove any `Manual Constraint` values. You can only remove the values that have been set earlier using the `Add Constraint Item` script. You can remove single or multiple items and months with this option. You can only remove constraint items from current and future months.

For any questions, please reach out to the GSheet Admin.
